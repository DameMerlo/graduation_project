# 2.技术栈

| 前端技术          | 说明                               |
| :---------------- | ---------------------------------- |
| Vue               | 前端框架                           |
| Vuex              | 全局状态管理框架                   |
| ElementUI         | 前端UI框架 【组件库】              |
| Axios             | 前端HTTP框架 【封装 处理异步请求】 |
| vue-element-admin | 项目脚手架 【开源脚手架】          |



| 后端技术     | 说明               |
| ------------ | ------------------ |
| SpringBoot   | 容器+MVC框架       |
| MyBatis      | ORM框架 【持久层】 |
| MyBatis-plus | MyBatis增强工具    |
| Redis        | 非关系型数据库     |
